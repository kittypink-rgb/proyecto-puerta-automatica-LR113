# Puerta Automática – Mini Proyecto Mecatrónica

Este repositorio contiene el desarrollo del mini proyecto de Mecatrónica
correspondiente a la asignatura Taller de Automatización 2 (Sección 1113).

## Integrantes
- Lady Sophia Joya Olivera
- Ruth Sinai Gomez Pacheco

## Descripción del proyecto
El proyecto consiste en el diseño de una puerta automática que detecta la
presencia de una persona mediante un sensor ultrasónico y controla su apertura
y cierre a través de un servomotor, integrando mecánica, electrónica y
programación.

## Contenido del repositorio
- **Codigo:** Programa en Arduino para el control del sistema.
- **CAD:** Diseño mecánico del sistema.
- **Documento:** Documento técnico del proyecto.
